📘 Student Record Manager
A Java console application to manage student records including adding, removing, and viewing student details.



📂 Project Structure

Copy
Edit
StudentRecordManager/
├── src/
│   └── com/
│       └── studentmanager/
│           ├── StudentManager.java
│           └── Student.java
✅ Features
Add a new student (name, age, roll number, branch)

Remove a student by name

View a specific student's details

View all student records

Exit the application
